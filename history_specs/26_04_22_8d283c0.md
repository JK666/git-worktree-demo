## 驗收標準

- 點擊語系切換按鈕，全站文案即時切換為英文/中文

## 技術約束

- 不引入 i18next 或任何外部 i18n library，純 React Context 實作
- `t()` 函式簽名：`t(key: string): string`，key 格式為 `section.field`（如 `hero.title`）
- data 檔改為 export 物件結構 `{ zh: '...', en: '...' }`，由 `t()` 依語系取值
- 不修改 CSS 與樣式相關檔案
