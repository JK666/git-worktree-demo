## 目標

Hero 區塊的 3 個統計數字（37%、2.4x、60%），在使用者捲動進入視窗時從 0 動畫跑到目標值，提升頁面視覺衝擊感。

- [x] 在 `src/components/Hero.jsx` 加入 `StatCounter` 元件：接收 `value` prop，內部使用 `useCountUp`，自動渲染動畫數字
- [x] 將 Hero stats 區塊的 `<span className="hero__stat-value">{stat.value}</span>` 替換為 `<StatCounter value={stat.value} />`

## 驗收標準

- 頁面載入後數字顯示為 0，捲入視窗後開始動畫跑到目標值
- 動畫時間約 1.8 秒，有明顯 ease-out 減速效果
- 三個數字格式正確：`37%`、`2.4x`、`60%`（小數位數與後綴保持一致）

## 技術約束

- 純 React hooks 實作（`useState`、`useEffect`、`useRef`），不引入任何外部 animation library
- 使用 `IntersectionObserver` threshold: 0.5 觸發動畫
- ease-out cubic 公式：`1 - Math.pow(1 - progress, 3)`
- 不修改任何 CSS 檔、data 檔或其他元件
- `parseStatValue` 解析規則：`/^([\d.]+)(.*)$/`，小數位數由原始字串決定
