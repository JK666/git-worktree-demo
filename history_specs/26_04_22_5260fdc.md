## 目標

在 Navbar 新增深色/淺色主題切換按鈕，使用者的偏好透過 localStorage 持久化，重新整理後主題不重置。

## 實作範圍

- [x] 在 `src/index.css` 的 `:root` 區塊後新增 `[data-theme="light"]` 覆寫所有顏色 CSS 變數（bg、text、border、card 等）
- [x] 在 `src/components/Navbar.jsx` 新增主題切換按鈕（☀️ / 🌙 圖示）
- [x] 建立主題初始化邏輯：頁面載入時讀取 `localStorage.getItem('theme')`，套用至 `document.documentElement`
- [x] 切換時更新 `data-theme` 屬性並寫入 `localStorage`
- [x] 在 `src/index.css` 新增切換按鈕的樣式（與 Navbar 風格一致）
- [x] 確認所有元件的顏色均使用 CSS 變數（不得有 hardcode 色碼）

## 驗收標準

- 點擊按鈕可在深色/淺色主題之間切換，全站顏色即時變化
- 重新整理頁面後，主題維持上次選擇
- 淺色主題下頁面可讀性良好（contrast ratio 符合 WCAG AA）
- 桌面與手機版 Navbar 均顯示切換按鈕

## 技術約束

- 不引入任何外部 theme library
- 純 CSS 變數實作，不使用 CSS-in-JS
- 使用 `document.documentElement.setAttribute('data-theme', ...)` 而非 class toggle
- 按鈕樣式需符合現有 Navbar 的 Glassmorphism 風格
- 不修改任何 data 檔（`src/data/`）
